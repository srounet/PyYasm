Python x86/x64 wrapper for Yasm

Using pyyasm
============

.. code-block:: python

    import pyyasm
    
    # some inline asm which does nothing just to show how the library works
    # __asm as to be bytes.
    __asm = b"""
	use32
	org 0
        pushfd
        pushad
        popad
        popfd
    """
    bytecode = pyyasm.assemble(__asm)
    print(bytecode)
    
    > b'f\x9cf`faf\x9d'
